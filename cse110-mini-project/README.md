# CSE110 Mini Project
## Bank Management System (Java)

This project is a console-based Bank Management System developed as part of the CSE110 course.  
It is designed to demonstrate Object-Oriented Programming (OOP) concepts in Java along with basic file handling.

## Features
- Create bank accounts with auto-generated unique account numbers
- Deposit and withdraw money
- View individual account details
- View all accounts
- Data persistence using file handling
- Custom exception handling

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Exception Handling

## Technologies
- Java
- File Handling
- Console-based UI
